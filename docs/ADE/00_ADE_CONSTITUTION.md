# Vision

Build an autonomous software engineering organism capable of understanding ideas, assessing them, designing solutions, generating production-quality software, validating itself, repairing itself, learning continuously, and evolving over time.

Portfolio Intelligence V4 is the first application built by this organism.

---

# Mission

Transform natural language ideas into production-ready software while maintaining architecture integrity, quality, documentation, testing, and continuous evolution.

---

# First Laws

## Law 1

Nothing shall be generated until the system understands why it should exist.

## Law 2

Everything generated must improve the organism.

## Law 3

Every improvement becomes reusable knowledge.

## Law 4

Improve the framework before improving applications whenever practical.

---

# Core Philosophy

Think before generating.

Reuse before creating.

Assess before implementing.

Validate before accepting.

Learn after every execution.
